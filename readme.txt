Project Title: AI Sentiment Analysis Program

Description:
This Python program is a proof of concept AI sentiment analysis tool. It allows users to input a URL, fetch the text content from the URL, and perform sentiment analysis on the text to determine if it is positive, negative, or neutral.

Installation:
- Ensure you have Python 3.x installed on your system.
- Install the required libraries using pip:
  pip install tkinter requests beautifulsoup4 textblob

Usage:
1. Run the program.
2. Enter a URL in the provided text entry.
3. Click the "Fetch Text" button to fetch the text content from the URL.
4. Click the "Analyze Sentiment" button to perform sentiment analysis on the text.
5. The program will display the sentiment analysis result in the GUI.
